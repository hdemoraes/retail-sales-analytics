# Retail Sales Analytics — Customer Segmentation & Prediction

## Overview
This project analyzes retail transaction data to identify revenue drivers, segment customers, and predict customer value.

## Key Features
- Data cleaning and feature engineering
- Exploratory data analysis (EDA)
- Customer segmentation (RFM-based)
- Revenue prediction using regression

## Results
- Revenue is highly concentrated among top customers
- Strong seasonal patterns (Nov–Dec peaks)
- Model achieved R² ≈ 0.48

## Tools
- Python (Pandas, Scikit-learn)
- Google Colab

## Business Impact
- Identify high-value customers
- Improve retention strategies
- Support data-driven marketing
